### Descripción del Proyecto
Este trabajo consiste en una aplicación de consola desarrollada en **C#** que permite calcular y mostrar una tabla de amortización de un préstamo con cuotas fijas (Sistema Francés). La aplicación utiliza la librería **Spectre.Console** para presentar los datos de forma profesional y clara.

#### Tecnologías aplicadas:
*   Tipos de datos (decimal, int)
*   Estructuras repetitivas (ciclo for)
*   Entrada y salida de datos por consola
*   Uso de librerías externas (NuGet - Spectre.Console)
*   Uso de funciones de la clase Math (`Math.Pow`, `Math.Round`)
